# **App Name**: Vani AI

## Core Features:

- Bilingual AI Player: Smart lecture playback with a tool for toggling AI-generated bilingual transcripts (e.g., Telugu-English) in real-time.
- Automatic Revision Notes: Generates structured summaries and exam-focused bullet points from lecture transcripts automatically.
- AI Doubt Tutor: A context-aware chat assistant that uses a tool to reference lecture materials to answer student queries in mixed languages.
- Smart Memory Flashcards: Automatically generates interactive study cards using spaced-repetition logic for long-term retention.
- Student Performance Hub: A centralized dashboard to track mastery of subjects, weak topics, and daily revision streaks stored in the database.
- Dynamic AI Quizzes: A tool that generates personalized assessments based on the student's least-understood lecture concepts.
- Identity & Sync: Cloud-based authentication for students to sync their preferences, college details, and learning styles across mobile devices.

## Style Guidelines:

- Primary Color: Vibrant Deep Orange (#FF6F1A) symbolizing energy and innovation, contrasting with a Navy Blue secondary tone for academic trust.
- Background Color: Soft Alabaster (#FAF9F8), a heavily desaturated version of the primary hue to reduce eye strain during long study sessions.
- Accent Color: Deep Berry Red (#E62E44), an analogous hue for highlighting errors, urgent alerts, and high-priority exam markers.
- Font pairing: 'Poppins' for high-impact geometric headlines and 'Inter' for highly legible, objective body text.
- Modern, soft-stroke dual-tone icons inspired by Apple's SF Symbols to maintain a premium, machined feel.
- Mobile-first, bottom-anchored navigation and rounded card system (16px - 24px radius) for an approachable, tactile UX.
- Subtle Duolingo-style physics for button presses and smooth layout transitions between study tabs.
