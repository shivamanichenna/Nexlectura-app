# Previewing Nexlectra

Follow these steps to start the development server.

1. **Start Next.js Dev Server**:
   ```bash
   npm run dev
   ```
   *The server will start on http://localhost:9002 (as configured in package.json).*

2. **Start Genkit Dev UI (Optional)**:
   ```bash
   npm run genkit:dev
   ```

3. **Open in Browser**:
   Navigate to http://localhost:9002 to view the landing page.
